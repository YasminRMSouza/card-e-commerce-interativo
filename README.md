👟 Interactive Product Card - Task #001

Este projeto é a resolução do primeiro ticket técnico de uma simulação de ambiente real de desenvolvimento Front-End. O objetivo foi criar um componente de card de e-commerce totalmente interativo, focando em manipulação de DOM e lógica de interface.

🚀 Funcionalidades

- **Seleção Dinâmica de Tamanhos:** Interface reativa que gerencia o estado "ativo" dos botões de numeração.
- **Controle de Quantidade:** Incremento e decremento com validação (mínimo de 1 unidade).
- **Cálculo de Preço em Tempo Real:** Atualização automática do subtotal conforme a quantidade selecionada.
- **Feedback de Compra:** Modal/Alerta consolidando os dados da seleção (quantidade, tamanho e valor total).
- **Layout Responsivo:** Design adaptável para diferentes tamanhos de tela utilizando Flexbox e CSS Grid.

🛠️ Tecnologias e Técnicas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização moderna, variáveis e efeitos de *hover*.
- **JavaScript (Vanilla):** Manipulação de eventos e lógica de negócio sem frameworks.
- **Clean Code:** Aplicação de nomes de variáveis semânticos, funções de responsabilidade única e organização modular do código.

🎨 Preview

- Interface limpa e intuitiva.
- Feedback visual imediato para ações do usuário.
